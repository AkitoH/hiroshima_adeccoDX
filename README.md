# 広島県 生成AI経営支援モデル実証 ／ 参加募集LP

令和8年度 広島県DX関連事業「生成AIを活用した経営支援モデルの構築に向けた実証業務」の、
**経営指導員向け 参加募集ランディングページ**です。GitHub Pages で公開します。

## 構成

| ファイル | 役割 |
|----------|------|
| `index.html` | 公開ページ本体（自己完結の静的HTML。外部依存はGoogle Fontsのみ） |
| `.nojekyll` | GitHub Pages の Jekyll 処理を無効化（静的HTMLをそのまま配信） |
| `経営支援モデル_募集LP.dc.html` | 元原稿（Claudeアーティファクト形式）。編集の起点として保管 |

## 公開手順（GitHub Pages）

1. このフォルダを GitHub リポジトリへ push
2. リポジトリの **Settings → Pages**
3. **Source** を `Deploy from a branch`、Branch を `main` / `/(root)` に設定
4. 数分後 `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開

## 編集メモ

- `index.html` を直接編集すれば内容を更新できます（FAQの開閉は素のJavaScriptで動作）。
- 「調整中」「確定後に掲載」表記の箇所は、県との協議・確定後に差し替えます。
